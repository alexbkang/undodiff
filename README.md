# undodiff

Neovim plugin that wraps the built-in `undotree-plugin` and `codediff.nvim` to show undo history as side-by-side diffs.

## Features
- Show undo history as side-by-side diffs for a single file.
- Optionally collapse unchanged lines for large blocks.
- Pass options through the plugin to configure `codediff.nvim`.
